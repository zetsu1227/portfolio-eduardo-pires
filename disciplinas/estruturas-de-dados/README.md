# 🧩 < Estruturas de Dados >
**Período:** 2025/2  
**Projeto:** Medição de Tempo em Jogos Digitais  
**Professor:** Dennis Giovani Balreira e Viviane Pereira Moreira

---

## 🎯 Problema a ser resolvido
Nesta cadeira, aprendemos sobre as principais estruturas de dados: listas, árvores e grafos.
O trabalho dado pelos professores tem como objetivo praticar a inserção e busca em árvores e as comparar por intermédio de um 
arquivo .csv que contém inúmeros jogos e suas tempos médios de jogatina dos usuários e um .txt que contém os jogos que o usuário
quer jogar; dessa forma, é preciso informar o tempo total médio para jogar todos os jogos que o usuário pede.

---

## 🏗️ Arquitetura e Tecnologias

- C(99) como linguagem principal
- CodeBlocks como ambiente de desenvolvimento
- Arquitetura baseada em Tipo Abstrato de Dados(TAD) com o objetivo de organizar as funções e estruturas
- Gerenciamento de inserção e de busca em arquivos .csv e .txt

---

## 🧱 Boas práticas aplicadas

- Versionamento e fluxo de desenvolvimento por meio do GitHub
- Uso de TADs e comentários nos códigos
- Chamadas do executável e suas entradas através da linha de comando
  
---

## 🤝 Soft Skills e Trabalho em Equipe

## 📈 Resultados e Aprendizados
- O que funcionou bem:  
  1. A AVL foi a estrutura que teve os melhores resultados de inserção e busca.
  2. O método de criar e chamar os executáveis por linha de comando dinamiza a correção e identificação de erros.
  3. O GitHub Desktop facilitou muito para a visualização e manipulação das versões.
  
- O que poderia melhorar:
  1. Criação de uma interface para o usuário afim de deixar mais simples o seu uso para não programadores. 
  
- Conceitos mais aplicados da disciplina:  
  1. Identificação da melhor opção de estrutura para resolver problemas específicos.
  2. Diferentes meios de representar as principais estruturas(lista, árvore, grafo).

- Lições para projetos futuros:  
  1. Focar em uso de árvores para armazenar e consultar dados várias vezes.
  2. Avaliar a necessidade do uso de algoritmos ótimos que possuem alto custo de processamento.
